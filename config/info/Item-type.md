# Tổng quan

Các lại Item gồm nhiều phương pháp lấy dữ liệu từ hệ thống của bạn. Mội lại item đi kèm với một Item key được hỗ trợ riêng và các thông số bắt buộc.

Các loại Item hiện đang được cung cấp bởi Zabbix:
* Zabbix agent checks
* SNMP agent checks
* SNMP traps
* IPMI checks
* Simple checks
    * VMware monitoring
* Log file monitoring
* Calculated items
* Zabbix internal checks
* SSH checks
* Telnet checks
* External checks
* Aggregate checks
* Trapper items
* JMX monitoring
* ODBC checks
* Dependent items
* HTTP checks
* Prometheus checks
